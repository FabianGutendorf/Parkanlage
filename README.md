# Parkanlage

Projekt für ein Client Server Beispiel.

Der Server übernimmt die Berechnung, während der Client den Aufruf der einzelnen Funktionen wahrnimmt.


#Flask installation:
sudo pip3 install Flask

#Virtuelle Umgebung
python3 -m venv venv
. venv/bin/activate

# Flask starten
export FLASK_APP=hello.py
flask run
